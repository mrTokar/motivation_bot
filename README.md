# Motivation Bot
Программа, которая поможет кураторам мотивировать студентов, рассылая им заранее подготовленные сообщения с изображениями.

### Установка
1. Склонируйте репозиторий:
   ```bash
   git clone
    ```
2. Установите необходимые зависимости:
    ```bash
    pip install -r requirements.txt
    ```
3. Настройте переменные окружения для доступа к API ВКонтакте:
Создайте файл `.env` в корневой директории проекта и добавьте следующие переменные:
    * VK_API_TOKEN - Ключ доступа сообщества API ВКонтакте (о том как получить в [документации](https://dev.vk.com/ru/api/access-token/getting-started?ref=old_portal#%D0%9A%D0%BB%D1%8E%D1%87%20%D0%B4%D0%BE%D1%81%D1%82%D1%83%D0%BF%D0%B0%20%D1%81%D0%BE%D0%BE%D0%B1%D1%89%D0%B5%D1%81%D1%82%D0%B2%D0%B0)).

### Использование
1. Подготовьте файл формата `.json` с id студентов и сообщениями, которые вы хотите отправить.
2. Запустите бота:
    ```cmd
    python main.py
    ``` 
3. Следуйте инструкциям в консоли для отправки сообщений студентам.

Программа использует файл формата `.json`, содержащий id студентов и сообщения, которые необходимо отправить.

```json
{
    "students": [
        {
            "vk_id": "123456789",
            "text": "Ваше мотивационное сообщение здесь."
        },
        {
            "vk_id": "987654321",
            "text": "Еще одно мотивационное сообщение."
        }
    ]
}
```
